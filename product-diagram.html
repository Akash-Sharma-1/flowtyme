Why plugin architecture
Built to last beyond any one tool
🔌
Edit YAML, not code
Enable a new source or calendar by editing one config file and restarting. Core engine untouched.
🔀
Multiple sources merge
Enable Notion + Obsidian simultaneously. All SourceItem[] flatten before slot assignment.
🤫
Secrets stay in .env
plugins.yaml never holds credentials. Tokens and passwords always in environment variables.
✦
AI-generated adapters
Describe a new source → /generate-parser writes the TypeScript plugin. Describe a calendar → /add-calendar-plugin does the same.
